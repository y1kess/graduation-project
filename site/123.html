About the Game

Valorant is a free-to-play competitive first-person shooter developed and published by Riot Games. It features intense tactical gameplay, precise gunplay, and a diverse roster of agents with unique abilities.

Game Modes

Unrated

Unrated mode is a casual game mode where players can play matches to practice and improve their skills without the pressure of ranked competition.

Competitive

Competitive mode is where players can test their skills in a ranked environment. Players earn and lose rank points based on their performance and can climb the ranks to prove their prowess.

Spike Rush

Spike Rush is a faster-paced game mode with a shorter match duration. Each player starts the round with the same randomly assigned weapon and abilities, adding an element of unpredictability.

Swiftplay

Swiftplay is a best-of-nine spike (Plant/Defuse) mode. It has most of the same gameplay rules as Unrated but with the following changes to adjust for its shorter duration: All player start each half with two points for their ultimate. No credit bonuses for loss streaks. Credits earned based on round outcome have adjusted values and are the same for both teams, with the exception of round 2

Deathmatch

Deathmatch is a multiplayer free-for-all game mode designed to allow players to hone their gunplay mechanics or warm up before playing Competitive. Each player loads into a random map with a random agent that they own. Players start in a brief warm-up phase before the real game starts. Players cannot buy or use abilities but are automatically given Heavy Shields. They can buy whatever weapons they want.

Escalation

Escalation is a 5v5 team deathmatch style mode where players kill and race the enemy team through a cycle of different weapons and abilities, advancing together as a team through Escalation levels. Players load into game with a random agent. The first team to finish level 12, or the furthest along in 10 minutes, wins the game.
